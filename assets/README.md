# Archivos de Audio

Para que el juego funcione correctamente, necesitas agregar los siguientes archivos de audio en esta carpeta:

1. **pasos.mp3** - Sonido corto que se reproduce cuando el jugador se mueve
2. **victoria.mp3** - Sonido que se reproduce cuando el jugador gana

Puedes descargar sonidos gratuitos de sitios como:
- freesound.org
- zapsplat.com
- soundbible.com

O crear tus propios archivos de audio.

## Especificaciones recomendadas:
- Formato: MP3
- Duración de pasos.mp3: 0.2 - 0.5 segundos
- Duración de victoria.mp3: 1 - 3 segundos
